mmdps.dms.mmdpdb module¶

This module deals with mmdpdb related actions.

mmdpdb is the database of MMDPS, containing 3 databased.
  1. SQLite stores meta-info like patient information, scan date, group relationships, research study cases and so on.
  2. MongoDB keeps all extracted features like networks and attributes.
  3. Redis is a high-speed cache that starts up upon request.
